### 관광 상황 상세

**1. 편의점에서 물건 사기**
- 첫 상황으로 적합 (짧고, 성공 경험)
- 대사 예: "これをください" → "袋いりますか?" → "いいえ、大丈夫です"

**2. 카페에서 주문**
- 사이즈, 테이크아웃 여부 묻는 흐름
- 대사 예: "アイスコーヒーください" → "サイズは?" → "Mで"

**3. 식당에서 주문**
- 메뉴 질문, 추천 요청, 계산
- 대사 예: "すみません、注文お願いします" → "おすすめは何ですか?"

**4. 역에서 표 사기**
- 목적지, 편도/왕복, 승강장 확인
- 대사 예: "渋谷までお願いします" → "片道ですか?" → "片道で"

**5. 길 묻기**
- 목적지 말하기, 방향 듣기
- 대사 예: "すみません、駅はどこですか?" → "まっすぐ行って、右です"

**6. 호텔 체크인**
- 예약 확인, 열쇠 받기
- 대사 예: "チェックインお願いします" → "お名前は?" → "〇〇です"

**7. 택시 타기**
- 목적지 전달, 도착 확인, 결제
- 대사 예: "東京駅までお願いします" → "ここで止めてください"

## 💼 비즈니스 (Business)

| # | 상황 | 장소 | 핵심 표현 | 난이도 |
|---|------|------|----------|--------|
| 1 | 공항 마중 | 공항 | はじめまして, お迎え | ★☆☆ |
| 2 | 명함 교환 | 회의실 | 名刺, よろしくお願いします | ★★☆ |
| 3 | 회의실 안내 | 사무실 | ご案内, こちらへ | ★★☆ |
| 4 | 택시 이동 | 택시 | 〜までお願いします | ★★☆ |
| 5 | 회의 중 기본 응답 | 회의실 | わかりました, 確認します | ★★☆ |
| 6 | 식사 접대 | 식당 | 乾杯, お口に合いますか | ★★★ |
| 7 | 작별 인사 | 현관 | お疲れ様, また | ★★★ |

### 비즈니스 상황 상세

**1. 공항 마중**
- 첫 만남 인사, 자기소개
- 대사 예: "〇〇様ですか?" → "はい、はじめまして" → "お迎えに参りました"

**2. 명함 교환**
- 명함 주고받기 예절
- 대사 예: "名刺をいただけますか?" → "こちらこそ、よろしくお願いします"

**3. 회의실 안내**
- 따라가기, 자리 안내
- 대사 예: "こちらへどうぞ" → "ありがとうございます"

**4. 택시 이동 (공유)**
- 관광 택시와 표현 공유, 격식만 다름
- 대사 예: "〜までお願いいたします"

**5. 회의 중 기본 응답**
- 이해했다, 확인하겠다 등
- 대사 예: "はい、わかりました" → "確認して、ご連絡いたします"

**6. 식사 접대**
- 건배, 음식 권유, 맛 묻기
- 대사 예: "乾杯!" → "お口に合いますか?" → "はい、とても美味しいです"

**7. 작별 인사**
- 감사, 다음 만남 기약
- 대사 예: "本日はありがとうございました" → "またお会いできることを楽しみにしています"

### 비즈니스 상황 → 일상 단어 매핑

| 상황 | 자연스럽게 배우는 단어 |
|------|----------------------|
| 공항 마중 | 인칭 (私/こちら), 시간 (お待たせ) |
| 명함 교환 | 호칭 (〜様/〜さん) |
| 회의실 안내 | 위치 (こちら/あちら), 층수 (〜階) |
| 택시 이동 | (관광과 공유) |
| 회의 응답 | 시간 (後で/すぐ), 빈도 (また) |
| 식사 접대 | 맛 표현 (美味しい/辛い), 양 (もう少し) |
| 작별 인사 | 시간 (今日/また) |

| 8 | 아르바이트 면접 | 매장/사무실 | 志望動機, シフト | ★★★ |

### 워홀/유학 상황 상세

**1. 슈퍼에서 장보기**
- 관광 편의점과 유사, 포인트카드 질문 추가
- 대사 예: "ポイントカードはお持ちですか?" → "いいえ、持っていません"

**2. 이웃에게 인사**
- 이사 인사, 간단한 자기소개
- 대사 예: "隣に引っ越してきました、よろしくお願いします"

**3. 우체국 택배**
- 택배 보내기, 소요 시간 확인
- 대사 예: "韓国に送りたいんですが" → "届くまで1週間くらいです"

**4. 휴대폰 개통**
- 플랜 선택, 신분증 제시
- 대사 예: "一番安いプランはどれですか?" → "身分証をお願いします"

**5. 병원에서 증상 설명**
- 증상 설명, 언제부터인지, 약 받기
- 대사 예: "どうしましたか?" → "頭が痛いです" → "いつからですか?" → "昨日からです"

**6. 은행 계좌 개설**
- 필요 서류, 도장
- 대사 예: "口座を開設したいんですが" → "届け印はお持ちですか?"

**7. 부동산 집 보기**
- 조건 설명, 비용 확인
- 대사 예: "駅から近いところがいいです" → "家賃は〇〇円で、敷金礼金は..."

**8. 아르바이트 면접**
- 지원 동기, 가능 시간
- 대사 예: "なぜこのお店で働きたいですか?" → "シフトは週に何回入れますか?"
