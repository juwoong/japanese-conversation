## 핵심 원칙 3가지

### 1. 여행 시뮬레이션이지, 공부가 아니다

사용자는 "일본어를 배우겠다"가 아니라 "여행에서 써먹을 말이 필요하다"는 동기로 온다. 모든 UI 판단 기준: **학습 앱에서 봤는가, 여행 가이드에서 봤는가?** 전자면 재고.

- 레슨 목록 → 여행 동선 지도
- 레벨/XP → "혼자 식당에서 주문할 수 있습니다"
- "틀렸습니다" → 점원이 자연스럽게 되묻기
- 복습 → 같은 장소의 새로운 상황

### 2. L1(한국어)은 경유지가 아니라 안전망

기본 화면에서 한국어를 빼되, 필요하면 꺼내볼 수 있게 둔다.

| 경유지 설계 (❌) | 안전망 설계 (✅) |
|:---:|:---:|
| 🔊 ふたり | 🔊 ふたり |
| 2명 ← 항상 보임 | [?] ← 탭하면 "2명" 3초 표시 |
| 매번 한국어를 읽게 됨 | 소리를 먼저 듣게 됨 |

**한국어 발음 표기(후타리, 이랏샤이마세 등)는 쓰지 않는다.** 한국어 음운 체계에 일본어 소리를 우겨넣으면 발음이 굳는다. ふ[ɸɯ]→"후"[hu], つ[tsɯ]→"쯔"[tɕu] 등. 소리 정보는 전부 네이티브 음성 재생으로 제공.

### 3. 듣기만 하다가 갑자기 말하게 하지 않는다

4단계 구조: **관찰 → 포착 → 참여 → 정리**. 포착(Catch)이 관찰과 참여 사이의 가교 역할. 첫 방문에서도 산출(말하기) 비율 40% 이상.

## 온보딩

### 목표

첫 실행 후 60초 안에 일본어 소리를 듣고 의미를 파악하는 경험. 설정 화면 없이 미니 체험으로 수준을 암묵 측정.

### 흐름

```
[앱 최초 실행]
    │
    ▼
┌─────────────────────────────────┐
│  전면 일러스트: 공항 도착        │
│  "도쿄에 도착했습니다."          │
│                                 │
│         [시작하기]               │
└─────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────┐
│  공항 안내 데스크 일러스트        │
│  직원 음성 🔊 + 인사 애니메이션  │
│                                 │
│  "따라 말해보세요"               │
│  [🎤 녹음] 또는 [듣기만 할게요]  │
└─────────────────────────────────┘
    │
    ├─ 음성 입력 시 → 발음 수준 측정 (내부)
    ├─ 스킵 시 → 보수적 초급 설정
    │
    ▼
┌─────────────────────────────────┐
│  직원이 출구 가리키는 애니메이션  │
│  "출구가 저쪽이래요.             │
│   첫 번째 일본어를 들었습니다!"  │
│                                 │
│       [도쿄 탐험 시작하기]       │
└─────────────────────────────────┘
```

**설계 포인트**:
- 온보딩에 일본어 자막 없음. 소리 + 그림 + 한국어 안내만. 첫 경험이 "읽기"가 아니라 "듣기"가 되도록
- "듣기만 할게요" 옵션 필수 — 공공장소, 심리적 부담 대응. 강제하면 이탈
- "레벨 테스트"라는 단어 없음 — "시험" 멘탈 모델 방지

### 매 세션 시작 시

```
┌──────────────────────────────────┐
│  지금 말할 수 있는 환경인가요?    │
│                                  │
│  [🎤 네, 말할 수 있어요]         │
│  [🔇 지금은 조용히 할게요]       │
└──────────────────────────────────┘
```

선택에 따라 산출 활동이 음성/탭 기반으로 분기.

| 순서 | 상황 | 핵심 목표 | 도입되는 공통 표현 |

| 1 | 공항 | 인사, 기본 질문 | すみません, ありがとう |
| 2 | 전철 | 숫자, 장소 묻기 | ～はどこですか |

| 4 | 편의점 | 물건 사기 | ～をください |
| 5 | 식당 | 주문, 계산 | ～にします, おかいけい |
| 6 | 관광지 | 사진 부탁, 길 묻기 | しゃしんをとってもらえますか |
| 7 | 쇼핑 | 사이즈, 면세 | ～はありますか |
| 8 | 긴급상황 | 도움 요청, 증상 | たすけてください, いたい |

### Soft Gating

모든 상황 열어두되, 시각적으로 권장 순서 유도.

- 완료: 컬러 아이콘 + ✓
- 권장 다음: 컬러 + 펄스 애니메이션
- 건너뛴 상황: opacity 0.6, 탭 가능
- 고급: opacity 낮음 + "공항, 전철을 먼저 해보는 걸 추천해요" 토스트

건너뛰면 해당 상황의 모델 대화에 선행 공통 표현을 자연스럽게 포함. 별도 "선행 학습" 화면 없음.

### 진행도 표시

- ⛔ "레벨 7", "1,240 XP"
- ✅ "혼자 전철을 타고, 편의점에서 물건을 살 수 있습니다"
